Сімейства шрифтів

Sans-serif

- шрифти без засічок, які зазвичай використовують для веб-сторінок, тому що вони добре читаються на екранах комп'ютера та мобільних пристроях. Наприклад: Arial, Helvetica, Verdana, Tahoma.

Serif

- шрифти з засічками, які зазвичай використовують для друкованих матеріалів, таких як книги та журнали. Наприклад: Times New Roman, Georgia, Baskerville.

Monospace

- шрифти, в яких кожен символ займає однакову кількість простору, що дозволяє легко вирівнювати текст, що містить пробіли або код. Наприклад: Courier New, Consolas, Lucida Console.

Cursive

- шрифти з написним почерком, які можуть бути використані для стилізації заголовків та логотипів. Наприклад: Brush Script, Comic Sans MS, Segoe Script.

Fantasy

- шрифти з фантастичними елементами, які можуть бути використані для стилізації заголовків та логотипів. Наприклад: Impact, Jokerman, Harrington.Fantasy - шрифти з фантастичними елементами, які можуть бути використані для стилізації заголовків та логотипів. Наприклад: Impact, Jokerman, Harrington.
